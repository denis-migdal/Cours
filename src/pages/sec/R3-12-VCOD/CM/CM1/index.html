On vient de calculer le syndrome. Codage: [ ID Q ] * [ I ] = [ C ] Décodage: [ Q ID ] * [ C' ] = [ I' S ] S est le syndrome. Si S = 0, pas d'erreurs (I=I'). Sinon, S indique la correction à appliquer à I' pour obtenir I. De manière générale, on pré-calcul un tableau donnant pour chaque syndrome possible, le correctif à appliquer. Dans notre cas,
Syndrome Position Masque
111 7 1 << 7 I = 10000000 ⨁ I'
110 6 1 << 6 I = 01000000 ⨁ I'
101 5 1 << 5 I = 00100000 ⨁ I'
100 4 1 << 4 I = 00010000 ⨁ I'
011 3 1 << 3 I = 00001000 ⨁ I'
010 2 1 << 2 I = 10000100 ⨁ I'
001 1 1 << 1 I = 00000010 ⨁ I'
